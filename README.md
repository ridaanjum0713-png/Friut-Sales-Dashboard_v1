# Fruit Sales Dashboard

Analyzed monthly fruit sales data using Excel and Google Sheets.

### Dashboard Includes:
- Sales comparison: June vs July
- MoM% Growth and Trend indicators
- Top/Bottom performing fruits
- Insights & Recommendations

### Key Findings:
- Overall sales dropped by 25.58%
- Guava was top performer with 133.33% growth
- Watermelon & Grape need inventory attention
